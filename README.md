# StudioData Lead Manager

Aplicație desktop macOS, locală și offline, pentru gestionarea lead-urilor StudioData.ro.

Datele sunt salvate în folderul standard `userData` al aplicației, nu în aplicația `.app`.

Comenzi utile:

```bash
pnpm install
pnpm run dev
pnpm run dist:mac
pnpm run dist:arm64
pnpm run dist:x64
pnpm run dist:universal
```
