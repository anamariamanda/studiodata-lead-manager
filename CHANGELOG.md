# Changelog

## v0.1.0 - Prima versiune funcțională

- Aplicație desktop locală pentru macOS, construită cu Electron.
- Gestionare lead-uri cu status, prioritate, sursă, contact, website și notițe.
- Auditor website integrat pentru briefuri simple, pe înțelesul clientului.
- Import contacte din fișiere RTF/TXT.
- Căutare în lista de lead-uri.
- Profil StudioData pentru servicii, ton, diferențiatori și semnătură.
- Generare email personalizat pe baza auditului și profilului StudioData.
- Trimitere email direct prin SMTP configurabil.
- Test SMTP în Setări > Email direct.
- Istoric activități pentru leaduri.
- Export/import backup și export CSV.
- Generare ofertă și export PDF.
- Build macOS cu compatibilitate minimă setată la macOS 10.15.
- Repository GitHub curățat și cod sursă urcat corect.
